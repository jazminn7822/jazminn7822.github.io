SHABR

This project was designed to use everything we've learned the past month from HTML and CSS to Bootstrap Version 4. The only thing we were given to create this website was the basic content or raw text like Home, Info, Pictures, Testimonials, Contact, etc. Our task was to practice using Bootstrap pieces like grid, content, components, utilities, etc, and to design our own unique website out of these Bootstrap pieces. This website had to be a Hot Air Balloon Website. I definetly faced some challenges along the way like placing the title "SHABR" and some other text to the right of an image. However, I was then able to solve it by looking at the Bootstrap documentation and W3.schools.

Before jumping into the project itself, I had to make a plan. Making a plan helped me out alot because I had created a sketch of how I wanted things to look, so when I actually stepped into writing the HTML and CSS for the website, I wasn't completely lost and I had an idea of where to start.
